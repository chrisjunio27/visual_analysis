This repository features a comprehensive Python script for conducting exploratory data analysis (EDA) of the Zara sales dataset using Matplotlib, Pandas, and more. The script helps reveal patterns, identify data issues, and extract actionable insights efficiently.

---

## üìä Key Features

- **Automatic Column Type Detection**  
  Smart detection of numerical vs. categorical columns for tailored exploration.

- **Basic Statistics & Data Info**  
  Prints descriptive statistics and a concise data overview to your console.

- **Visualization Suite:**  
  The script generates a set of clear, professional charts:
  - **Distribution Histogram**  
    Visualizes how numerical data is distributed.
  - **Box Plot**  
    Detects outliers and visualizes variable spread.
  - **Missing Values Bar Chart**  
    Quickly identifies columns with data quality issues.
  - **Correlation Heatmap**  
    Reveals relationships and dependencies.
  - **Categorical Distribution**  
    Shows most frequent values per category.
  - **Trend Line Plot**  
    Visualizes patterns and changes over records, e.g. sales over time.
  - **Scatter Plot**  
    Examines relationships between two variables.

- **Insights Summary**  
  Highlighted key findings from your EDA.

- **Professional Visualization Layout**  
  Clean, multi-panel subplot arrangements for impact and clarity.

- **Resilient Missing Value Handling**  
  Manages incomplete data gracefully and robustly.

## üöÄ Getting Started

1. **Clone this repository**
   ```bash
   git clone https://github.com/chrisjunio27/visual_analysis.git
   cd visual_analysis
   ```

2. **Install dependencies**
   ```bash
   pip install matplotlib pandas numpy seaborn
   ```

3. **Run the analysis**
   - Edit the script with your Zara sales CSV file path.
   - Run:
     ```bash
     python your_eda_script.py
     ```
